# Тестовое задание по аггрегации данных из MongoDB
## Запуск приложения
Установите все зависимости
```bash
poetry install
```
В корневой папке проекта добавьте .env файл, с данными полями:
```
MONGO_HOST
MONGO_PORT
MONGO_NAME
MONGO_COLL
TOKEN
```
Для запуска используйте
```bash
poetry run python3 src/main.py
```
